==============================================================================
collective.printrss
==============================================================================

``collective.printrss`` let you to print a rss feed from a given URL for `Plone`_ website.

Features
--------

- Can be print a rss feed.

Installation
------------

Install collective.printrss by adding it to your buildout::

    [buildout]

    ...

    eggs =
        collective.printrss


and then running ``bin/buildout``
